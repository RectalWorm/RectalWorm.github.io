Introducing EZ Lora Downloader.

What is the most inconvenient thing while using WebUI? 

When creating images by referencing other people's work, it's not easy to download the Lora file that the person used without knowing downloadable or not.


With EZ Lora Downloader, you can solve this problem at once.

Here's how it works:

  1. Select the image you want to replicate 

  2. Send to T2I or Send to I2I(Copying generation data is also an option.)

  3. Click "Search LoRa".

  4. EZ Lora Downloader will display the downloadable Lora files that you don't have.

  5. Download the one you need.

  6. Refresh and Generate!

You can browse various pictures that you might want to reproduce on the Feed tab.

  1. Newest : Real-time images by WebUI users

  2. Popular : Most popular pictures

  3. Favorite : Pictures that you have liked (with a recommendation feature to be added in the future).

  4. Customize your feed! Change your nickname and promote yourself.

  • In the future, we plan to improve the extension so that you can download missing items such as Models, Samplers, and other extensions.

Installation

If downloading directly:

  1. Create a folder named "sd_feed" in the "extensions" folder within the WebUI, and extract the compressed file there.

If installing from within the WebUI:

  1. Click the "install from url" button in the "extensions" tab.

  2. Enter https://github.com/todhm/sd_feed.

  3. Click the "install" button.

If cloning from Git:

  1. In the terminal, navigate to the "extensions" folder and enter the command "git clone https://github.com/todhm/sd_feed.git".
