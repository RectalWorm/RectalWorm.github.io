Read Invisible Prompt On Civitai

Before 2 months ago,i reported this bug to civitai

https://civitai.com/questions/349/why-cant-the-prompt-automatically-read-uploaded-images-now

but they dont keep more attention on it.

So i made this guide for Read Invisible Prompt On Civitai

1、Open your browser, such as Chrome, Firefox, Safari or Edge.

I use Chrome for example.

2、Go to the official website of the user script manager, such as Tampermonkey or Greasemonkey.

I installed Tampermonkey

3、Open this script and install it.

https://greasyfork.org/scripts/459579-stable-diffusion-image-metadata-viewer

4、Find a image on civitai,even it cant show any meta data.

For instance, I'll use the first cover of this model as an example.

5、Right click image and choose open with new tab

also you can choose copy the image address and open the new tab to browse it.

https://image.civitai.com/xG1nkqKTMzGDvpLrqFT7WA/8af758cc-37c2-48ca-aa41-4729db53eec0/width=1840/grid-0001-1.jpeg

5.5 this BUG is fixed,dont need new extension for chrome.

Please update to version 0.24 over~

5. 5、If you cant see button in this example, it seems that latest browser Content-Security-Policy blocked script.

this is a BUG for script with lastest browser, wait to fix or use this extension to close CSP in png page.

https://chrome.google.com/webstore/detail/disable-content-security/ieelmcmcagommplceebfedjlakkhpden

Install this↑ and click extension button then background will be black and script button will show.

6、Left-click the button in the top left corner, and prompt you want to know will be displayed.

7、if you cant find button, it seems that this image dont have extra prompt.
